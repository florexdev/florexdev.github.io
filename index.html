// 01 — hakkımda
Ben Kimim

Merhaba! Ben florexdev, Bilecik Şeyh Edebali Üniversitesi Söğüt MYO'da Bilgisayar Programcılığı okuyorum.

Frontend geliştirme ve masaüstü uygulama geliştirme alanlarında kendimi geliştiriyorum. HTML, CSS, JavaScript ve jQuery ile modern web arayüzleri; C# ile masaüstü uygulamalar geliştiriyorum.

PHP ile sunucu tarafı, Python ile scripting ve otomasyon konularında da çalışmalarım bulunuyor. BTK Akademi'den 2 sertifikam var. Şu an staj deneyimi kazanmak istiyorum.

Tarayıcı yer imlerini görsel kartlara dönüştüren web aracı. URL girildiğinde sitenin başlığını, açıklamasını ve küçük resmini otomatik çeker. Pinterest benzeri grid tasarımıyla düzenler.
